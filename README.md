# ✈️ AI Agentic Travel Planner

An AI-powered travel planner using LangGraph and Ollama.

## 🚀 Features
- Multi-agent system (Flights, Hotels, Itinerary, Budget)
- Local LLM using Ollama (no API cost)
- Streamlit web interface
- Chat-based interaction

## 🛠 Tech Stack
- Python
- LangGraph
- Ollama
- Streamlit

## ▶️ Run Locally

```bash
pip install -r requirements.txt
ollama run llama3
streamlit run app.py
